## 🚀 Sobre o Projeto
O **SnapNoMark** é uma plataforma de alta performance desenvolvida para usuários que buscam praticidade e qualidade extrema. O foco do projeto é remover a marca d'água (logo) dos vídeos do TikTok em tempo real, entregando o arquivo MP4 original em HD (Alta Definição).

O site foi construído com uma filosofia **Mobile-First**, garantindo que usuários de iPhone e Android tenham a melhor experiência de download diretamente pelo navegador, sem necessidade de instalar aplicativos pesados.

---

## ✨ Funcionalidades Principais
- ✅ **Remoção de Marca d'Água:** Motor inteligente que limpa o vídeo automaticamente.
- ✅ **Qualidade HD:** Download preservando a resolução original do conteúdo.
- ✅ **Downloads Ilimitados:** Sem barreiras de uso ou cobranças ocultas.
- ✅ **Notificações Push:** Sistema de alertas para atualizações e novos recursos via Service Worker.
- ✅ **Interface Neon PRO:** Design escuro moderno otimizado para telas OLED de smartphones.

---

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** HTML5, CSS3 e JavaScript (ES6+).
- **Estilização:** [Tailwind CSS](https://tailwindcss.com/) (Design Responsivo e Rápido).
- **Processamento:** Integração com Rest API (RapidAPI) para extração de dados.
- **Hospedagem:** [Vercel](https://vercel.com/) (Edge Computing para baixa latência).
- **Análise:** Google Analytics 4 (Monitoramento de tráfego em tempo real).
- **Monetização:** Monetag (Smart Tags e Notificações Push).
